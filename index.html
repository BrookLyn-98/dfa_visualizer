Automaton Visualizer

Load a graph and press Start.

How to Use

  • Paste a JSON config describing your automaton in the Simulator tab.
  • Click Update Graph to render it on the canvas.
  • Enter a test string and click ▶ Start Simulation to animate the run.
  • Drag any state node on the canvas to reposition it — the JSON updates automatically.

JSON Format

  • states — list of all state names
  • initial — the start state
  • finals — list of accepting states
  • delta — transition map: state → symbol → [targets]
  • layout — optional x/y (0–1) positions per state
  • Use "" as a symbol for λ-transitions (NFA only)
